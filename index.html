About this project

CatDoku is an interactive way of learning sudoku while having hard puzzles to play with to practice and dominate the tecniques you're currently learning. All this while seeing cute little cats! (soon).

What are the rules ?

  • Sudoku grid consists of 9x9 spaces.
  • You can use only numbers from 1 to 9.
  • Each 3×3 block can only contain numbers from 1 to 9.
  • Each vertical column can only contain numbers from 1 to 9.
  • Each horizontal row can only contain numbers from 1 to 9.
  • Each number in the 3×3 block, vertical column or horizontal row can be used only once.
  • The game is over when the whole Sudoku grid is correctly filled with numbers.

One aditional thing to have in mind is that every rule that applies for a column or row also applies for a block. This is extremly used in strategies we will see together with cats :3
